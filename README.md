# 居合道 — Тренажёр по теории иайдо

Приложение для самоподготовки к экзаменам по теории иайдо (уровни 5→1).

---

## Быстрый старт

### 1. Установи Python

Нужен Python версии 3.10 или новее.

- Скачай: https://www.python.org/downloads/
- При установке **обязательно** поставь галку **"Add Python to PATH"**

### 2. Запусти тренажёр

Дважды кликни на файл **`start.bat`**

Откроется чёрное окно, установятся зависимости (~10 секунд), затем автоматически откроется браузер на адресе `http://localhost:5000`.

> Если браузер не открылся сам — открой его вручную и перейди на `http://localhost:5000`

### 3. Останови

Закрой чёрное окно с сервером (или нажми `Ctrl+C` в нём).

---

## Разделы

| Раздел | Что делает |
|--------|-----------|
| **Изучение** | Карточки с переворотом. Выбери уровень, листай термины, отмечай «Знаю / Повторить». |
| **Самоподготовка** | Тест с вариантами ответа. Выбор уровня, режим слабых мест, статистика. |
| **Тестирование** | Открытые вопросы. Ответы отправляются на почту проверяющему. |
| **Настройки** | Email проверяющего, загрузка новой таблицы терминов. |

---

## Загрузка своей таблицы терминов

Открой **Настройки → Таблица терминов**.

Перетащи туда файл `.xlsx` или `.csv` с колонками:

| term | level | definition |
|------|-------|------------|
| Аюмиаси | 5 | переставной шаг |
| Дза-ката | 5 | метод посадки |

Старая таблица сохранится как резервная копия (`questions.json.bak`).

---

## Настройка отправки тестов на почту

Используется Gmail. Нужен **пароль приложения** (не обычный пароль):

1. Войди в аккаунт Google
2. Перейди: **Google Аккаунт → Безопасность → Двухэтапная аутентификация → Пароли приложений**
3. Создай новое приложение, скопируй 16-значный код
4. В тренажёре: **Настройки** → введи email и этот код

## Устранение проблем

**«Python не найден»** — убедись что при установке Python была отмечена галка "Add Python to PATH". Переустанови если нужно.

**Браузер не открывается** — открой вручную: `http://localhost:5000`

**Порт занят** — если порт 5000 уже используется другим приложением, измени в `server.py` последнюю строку: `app.run(debug=True, port=5001)` и открывай `http://localhost:5001`

**Ошибка при загрузке xlsx** — убедись что в чёрном окне установился `openpyxl` без ошибок. Если нет — запусти вручную: `pip install openpyxl`
