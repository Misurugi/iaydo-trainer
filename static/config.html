Gmail отправителя

Для отправки через Gmail нужен пароль приложения — не обычный пароль аккаунта.
Google Аккаунт → Безопасность → Двухэтапная аутентификация → Пароли приложений. Создай новое приложение, скопируй 16-значный код.

Таблица терминов

Загрузи .xlsx или .csv с колонками term / level / definition (или на русском: термин / уровень / определение — колонка «уровень» = кю-ранг цифрой от 1 до 5). Текущая таблица будет заменена, старая сохранится как резервная копия.
